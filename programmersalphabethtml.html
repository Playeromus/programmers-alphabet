HTML







!DOCTYPE html


!DOCTYPE must always be used at the start. The purpose of this declaration is to help the site display correctly.

An example of using !DOCTYPE html:

    <!DOCTYPE html>
    <html>
    ...




html


html is an element that must be used at the start. It is the root element, which in simple words, means that it holds all the next code together.

An example of using html:

    <!DOCTYPE html>
    <html>
    ...
    </html>




head


The head is used to contain the meta information about the HTML page. Which can be used to dictate that all the next elements that you input in the head, will be used in the same way for all other instances.

An example of using head:

    <!DOCTYPE html>
    <html>
    <head>
    <style>
    pre {
    font-size:50px;
    text-align:center;
    }
    </style>
    </head>
    ...
    </html>




title


The title can only be used in the head element. The title element gives your HTML webpage a title.

An example of using title:


    <!DOCTYPE html>
    <html>
    <head>
    <title>Programmers Alphabet</title>
    ...
    </html>




body


The body element defines the webpage's body. Whatever code you put inside the body, will show up on the webpage.

An example of using body:


    <!DOCTYPE html>
    <html>
    <body>
    <h1>The element body</h1>
    </body>
    </html>




h1 - h6


The h elements creates a heading. The numbers indicate the size., the 1 is the biggest while the 6 is the smallest.

An example of using the h1 - h6 element:


    <!DOCTYPE html>
    <html>
    <body>
    <h1>The h1 - h6 elements</h1>
    </body>
    </html>




p


The p element creates a paragraph. You cannot make the paragraph bigger by putting a number next to it like the h1 - h6 elements.

An example of using the p elements:


    <!DOCTYPE html>
    <html>
    <body>
    <p>The p elements</p>
    </body>
    </html>
